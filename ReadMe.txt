Если при запуске\компилляции появляются ошибки:

1)Заходим в менеджмент nuGet пакетов
	http://screencast.com/t/q0q6VfdZuJp2

2)Выбираем в фильтре Обновление -> Все пакеты, 
после непродолжительного поиска появляется кнопка Обновить [Update]
Обновляемся
	http://screencast.com/t/GyPktBET

3)Если возникли ошибки, нажимаем продолжить\Ок.
Если устарели компоненты студии, она сама откроет окно обновления компонентов.
По нажатию на кнопку Обновить [Update] в броузере откроется страница загрузки. 
Ничего не нажимаем (Там есть манящая кнопка "Скачать". она грузит демки проектов, которые нас не интересуют)
Через несколько секунд начнется автоматическая загрузка компонента.
Устанавливаем компоненты.
Если запущена студия - выключаем.
	http://screencast.com/t/piZ65PtAbfvT

4)После установки запускаем nuGet и продолжаем обновление пакетов(пункты 1 и 2)
В итоге получаем отсутствие элементов для обновления и приступаем к работе с проектом.
	http://screencast.com/t/aFzvQVO8j7u
